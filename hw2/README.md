Configuration
=============
The following environment variables must be set:
* AWS_ACCESS_KEY_ID
* AWS_SECRET_ACCESS_KEY
* ELASTIC_SEARCH_HOST
* ELASTIC_SEARCH_INDEX

To Run
======
```
$ mvn clean package tomcat7:run
```
